## Getting the files

<br>

Clone the project repository running in your terminal:

```
git clone https://github.com/marcus88santos/squad16-engplay.git
```
<br>

Enter project folder:

```
cd Squad16-EngPlay
```

Download dependencies:

```
npm install
```

Run application:

```
npm start
```
